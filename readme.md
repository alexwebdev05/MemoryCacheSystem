# Memory cache system - Proyecto de aprendizaje
Este proyecto forma parte de mi aprendizaje de Rust. Por el momento está en desarrollo.

## Funcionalidades
Esta aplicación permite hacer un caché parecido al de Redis en el cual puedes almacenar valores asociados a una key.

## Instalación
Ya que no ha sido lanzada la v1.0 habrá que hacer la siguiente instalación:

```bash
git clone https://github.com/alexwebdev05/ITTaskManager.git
```

- Solo correr el proyecto:
```bash
cargo run
```

- Compilar el binario:
```bash
cargo build
```

El binario se almacenará en `./target/debug/memory_cache_system.exe`
